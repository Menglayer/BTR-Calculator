# Bitlayer 空投计算器 · 使用教程

> 深色 + 橙色 UI，支持移动端。结果区默认 **关注 @menglayer 后解锁**。

## 🚀 快速开始（3 步）
1. **打开**
2. **关注并解锁**：点右侧结果区的 “去关注 @menglayer” → 返回页面点 **“我已关注，显示结果”**（本地通过 `localStorage` 记忆状态）。  
3. **填 4 个输入**，右侧实时给出你的空投估值：  
   - **宝石数量**（💎 按默认 **$0.0025/颗** 计，可在高级设置修改）  
   - **积分数量**（你的账户积分）  
   - **BTR 数量**（用于参与/锁仓，参与 BTR 奖励池分配）  
   - **赛车奖励（USD）**（🏁 直接填你拿到的美元金额）

---

## 🧮 结果面板怎么读
- **总空投池（估值）**：基于 `FDV × 空投比例` 计算得到的总额。  
- **三池拆分**：积分池 / BTR 池 / 赛车池（权重可在高级设置里改，自动归一化）。  
- **1 积分的预测价值**：`积分池 ÷ 全网积分总量`。  
- **宝石单价**：展示当前用于计算的宝石单价。  
- **我的空投估值**：来自 4 部分的合计：
  - 积分贡献（与你的 **积分** 成正比）  
  - BTR 贡献（与你的 **BTR/全网 BTR** 占比成正比）  
  - 赛车贡献（等于你输入的 **赛车奖励 USD**）  
  - 宝石价值（= **宝石数量 × 宝石单价**）  
- **我的非宝石空投占比**：`(积分贡献 + BTR 贡献 + 赛车贡献) ÷ 总空投池`。

---

## ⚙️ 高级设置（可选）
> 展开右上角 **“高级设置”** 调整全局参数，影响所有结果：

- **TGE FDV**：默认 `200,000,000`（USD）。  
- **空投比例**：默认 `10%`。  
- **全网积分总量**：默认 `285,000,000`。  
- **宝石单价**：默认 `$0.0025/颗`（右侧会完整显示精度）。  
- **三池权重**：默认 `积分 70% / BTR 20% / 赛车 10%`（和自动归一化，不必刚好等于 100%）。  
- **全网 BTR 参与量**：用于算你的 BTR 池占比，默认 `100,000,000`。

---

## 🧠 计算公式（公开透明）
- `AirdropPoolUSD = FDV × (空投比例%)`  
- `积分池 = AirdropPoolUSD × 积分权重`  
- `BTR 池 = AirdropPoolUSD × BTR 权重`  
- `赛车池 = AirdropPoolUSD × 赛车权重`  
- `每分价值 = 积分池 ÷ 全网积分总量`  
- `我的积分贡献 = 我的积分 × 每分价值`  
- `我的 BTR 占比 = 我的 BTR ÷ 全网 BTR 参与量`  
- `我的 BTR 贡献 = BTR 池 × 我的 BTR 占比`  
- `我的赛车贡献 = 你输入的赛车奖励（USD）`  
- `我的宝石价值 = 宝石数量 × 宝石单价`  
- `我的总估值 = 积分贡献 + BTR 贡献 + 赛车贡献 + 宝石价值`  
- `我的非宝石空投占比 = (积分贡献 + BTR 贡献 + 赛车贡献) ÷ AirdropPoolUSD`

## 📝 示例操作（建议）
1. 在左侧依次填：宝石数量、积分数量、BTR 数量、赛车奖励（USD）。  
2. 如需模拟不同方案：在 **高级设置** 里微调 `FDV/空投比例/三池权重/全网 BTR`，右侧结果会即时变化。  
3. 想只看“纯空投”（不含宝石），把“宝石数量”留空即可；或在结果区查看“我的非宝石空投占比”。

## ❓ 常见问题（FAQ）
**Q1：为什么看不到结果？**  
A：首次进入需要关注后解锁。点击右侧弹窗的 **“去关注 @menglayer”**，回到页面点 **“我已关注，显示结果”**。状态存于 `localStorage`，清除浏览器数据后需重新解锁。

**Q2：赛车奖励为什么是 USD？**  
A：为简化使用，赛车部分直接输入你最终拿到的美元金额，和三池权重互不冲突（权重只影响信息展示）。

**Q3：宝石价格能改吗？**  
A：可以，在「高级设置 → 宝石单价」修改，右侧会以 **高精度** 显示。

**Q4：我想恢复默认？**  
A：点右上角 **“重置”**。

**Q5：小数支持吗？**  
A：支持。输入支持任意精度（浏览器允许范围内）。

**Q6：移动端可用吗？**  
A：支持响应式布局，移动端也可流畅使用。
